正版 Java 版《我的世界》

本服使用正版验证登录，需要 Minecraft Java 版正版账号。基岩版（手机／主机版）无法进入。

客户端版本 1.21.11

在启动器中选择 1.21.11 版本启动。可以使用 OptiFine、Sodium 等优化模组，但请勿使用任何作弊类模组或材质。

进服第一件事：阅读服规

出生点备有《服规与指令》，请务必读完——它不只是规则，也是成为正式玩家、解锁完整权限的必经之路。也可以先在本站 服规页面 阅读。
